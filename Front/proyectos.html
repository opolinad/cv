Videogames app

Desarrollo de una app en la que se trae información de una API sobre videojuegos. El back de la aplicación consiste en levantar un servidor el cual recibe la información y descarta información no relevante para el front. Además de esto se crean rutas para hacer consultas HTTP y se levanta una base de datos en la que se almacena tanto información de la API como información que el usuario provee (creación de registros de videojuegos). El front es un SPA y se compone de 4 rutas diferentes: landing page, página principal, página de creación de registros de videojuegos y página donde se muetsra el detalle de un videojuego. Este proyecto se hizo con las siguientes tecnologías: CSS puro, React, Redux, PostgreSQL, Sequelize, Express, Google Fonts y JavaScript.

GraphQL playground de libros

Desarrollo de un servidor en Apollo para consultas con GraphQL. La interfaz de usuario (el playground de consultas) es de Apollo. Este proyecto cuenta con información de la API del New York Times, con la cual se alimenta una base de datos Mongo (proveedor Atlas). Para llenar la base de datos se tomó información de los Best-Seller y se mezcló información de autores, editoriales y libros. El lenguaje en el que fue desarrollado el proyecto es JavaScript, además se usó JWT para autorizar las consultas de autores y las mutaciones. Por último para manipular la base de datos se hizo uso de Mongoose.
